## 一、项目概述

Hermes Agent 是由 **Nous Research** 开发的自改进 AI Agent。与传统静态 Agent 不同，它具有以下核心特性：

1. **自主学习能力** - 从任务中自动创建新技能
2. **技能自我改进** - 在使用过程中不断完善
3. **周期性自省** - 定期进行自我复盘
4. **跨会话记忆** - 搜索历史对话，构建用户模型

## 五、与 OpenClaw 的关系

Hermes Agent 提供了从 OpenClaw 迁移的完整支持：

```bash
hermes claw migrate              # 交互式迁移
hermes claw migrate --dry-run    # 预览迁移内容
hermes claw migrate --preset user-data   # 无密迁移
```

迁移内容：
- SOUL.md 个性化文件
- MEMORY.md 记忆
- USER.md 用户信息
- 自定义技能
- API 密钥（Telegram、OpenRouter、OpenAI、Anthropic 等）
- 消息网关配置
